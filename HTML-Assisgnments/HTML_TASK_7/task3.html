<!DOCTYPE html> (Tells browser you are using HTML5)

<html lang="en"> (Root element of HTML document)

    <head> (Metadata of document)

        <meta charset="UTF-8"> (Character encoding)

        <meta name="viewport" content="width=device-width, initial-scale=1.0"> (Responsive viewport)

        <title> HTMLTASK-7 </title> (Defines page title)

    </head>

    <body> (Content of the page)


        <p>This is Paragraph Tag</p> (Represents a paragraph)

        <b>This is Bold Text</b> (Makes text bold)

        <strong>This is Strong Text</strong> (Important bold text)

        <i>This is Italic Text</i> (Makes text italic)

        <em>This is Emphasized Text</em> (Emphasized text, usually italic)

        <u>This is Underlined Text</u> (Adds underline)

        <mark>This is Highlighted Text</mark> (Highlights text)

        <small>This is Small Text</small> (Makes text smaller)

        <del>This is Deleted Text</del> (Strikethrough for deleted text)

        <ins>This is Inserted Text</ins> (Inserted text with underline)

        <sub>This is Subscript</sub> (Subscript text, e.g., H2O)

        <sup>This is Superscript</sup> (Superscript text, e.g., X2)

        <q>This is Short Quotation</q> (Inline quotation)

        <blockquote>This is Blockquote</blockquote> (Long quotation block)

        <abbr title="HyperText Markup Language">HTML</abbr> (Abbreviation with tooltip)

        <cite>This is Citation</cite> (Cites a title of work)

        <code>This is Inline Code</code> (Represents inline code)

        <pre>This is Preformatted Text</pre> (Shows preformatted text)

        <address>This is Address</address> (Contact info)


    </body>
</html>
