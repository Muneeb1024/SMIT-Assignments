What is HTML?

HTML (HyperText Markup Language) is the standard language used to create structure of web pages.HTML is the backbone of every webpage.Every website you see on the internet is built using HTML in combination with CSS and JavaScript.

HTML Boilerplate?

Boilerplate in HTML is the basic starter code (Standard Structure) that every HTML document begins with. 


<!DOCTYPE html>               → Declares the HTML version (HTML5) to the browser
<html lang="en">              → Root element, 'lang' defines the language of the page

<head>                        → Contains metadata, title, and links (not visible on the page)
  
    <title>Document</title> → Title of the page (shown in browser tab)

</head>

<body>                    → Contains all visible content of the web page

    Your content goes here

</body>

</html>


What is Webpage?

A webpage is a single page of Website. like Signup Page, Login Page, Home Page, Product Page, Product Categories Page etc.

What is Web Site?

Website is a collection of all related webpages linked together under a common domain.

What is Tags?

HTML tags are like containers that hold content and tell the browser how that content should be displayed.

What is Elements?

An HTML element is a complete structure made of a tag (or tags) and the content inside it.
