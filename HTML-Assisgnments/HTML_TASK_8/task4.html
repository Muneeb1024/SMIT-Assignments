Task:4 (List of Semantic and Non-Semantic Tags)

Semantic Tags

  • <header> - Defines the header section
  • <nav> - Navigation links
  • <main> - Main content of the page
  • <section> - A section in a document
  • <article> - Independent content (like blog post)
  • <aside> - Side content
  • <footer> - Defines footer section
  • <figure> - For images/diagrams with caption
  • <figcaption> - Caption for figure
  • <mark> - Highlighted text

Non-Semantic Tags

  • <div> - Generic container
  • <span> - Generic inline container
  • <b> - Bold text (without importance)
  • <i> - Italic text (without importance)
  • <u> - Underlined text
  • <font> - Deprecated font styling
  • <center> - Deprecated center alignment
  • <big> - Deprecated bigger text
  • <small> - Smaller text
  • <s> - Strikethrough text
