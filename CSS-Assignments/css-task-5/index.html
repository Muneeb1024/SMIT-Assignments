background-color

Sets the background color of an element.

Roman Urdu: Element ka peecha rang set karta hai. Jaise red, blue, green ya koi bhi color.

background-color: #ff6b6b; background-color: rgb(255, 107, 107); background-color: rgba(255, 107, 107, 0.8); background-color: hsl(0, 100%, 71%); background-color: transparent;
Background Color Demo

background-image

Sets one or more background images for an element.

Roman Urdu: Element ke peeche photo ya design lagata hai. Gradient colors bhi kar sakte hain.

background-image: url('image.jpg'); background-image: linear-gradient(45deg, #667eea, #764ba2); background-image: radial-gradient(circle, #ff6b6b, #4ecdc4); background-image: url('image1.jpg'), url('image2.jpg');
Background Image Demo

background-repeat

Controls how background images are repeated.

Roman Urdu: Photo repeat karna hai ya nahi? Sirf horizontal, vertical ya bilkul nahi repeat.

background-repeat: repeat; /* Default */ background-repeat: no-repeat; /* No repetition */ background-repeat: repeat-x; /* Horizontal only */ background-repeat: repeat-y; /* Vertical only */ background-repeat: space; /* Evenly spaced */ background-repeat: round; /* Scaled to fit */
No Repeat Demo

background-position

Sets the position of background images.

Roman Urdu: Photo kahan rakhna hai? Top, center, bottom, left, right - jahan chahiye wahan.

background-position: top left; /* Keywords */ background-position: center; /* Center both axes */ background-position: 50% 50%; /* Percentages */ background-position: 10px 20px; /* Length values */ background-position: right 10px bottom 20px; /* Edge offsets */
Position: Top Right

background-size

Controls the size of background images.

Roman Urdu: Photo kitna bara ya chota karna hai? Cover (pura cover), contain (fit) ya custom size.

background-size: auto; /* Original size */ background-size: cover; /* Cover entire element */ background-size: contain; /* Fit within element */ background-size: 100px 200px; /* Specific dimensions */ background-size: 50% 25%; /* Percentage values */
Size: 100px √ó 100px

background-attachment

Controls whether background images scroll with content or remain fixed.

Roman Urdu: Scroll karne pe photo saath move ho ya fixed rahe? Fixed means hamesha same jagah.

background-attachment: scroll; /* Default - scrolls with content */ background-attachment: fixed; /* Fixed to viewport */ background-attachment: local; /* Scrolls with element content */
Fixed Attachment Demo (scroll to see effect)

background-origin

Specifies the origin position of background images.

Roman Urdu: Photo kahan se start hona chahiye? Border se, padding se ya content area se?

background-origin: border-box; /* From border edge */ background-origin: padding-box; /* From padding edge (default) */ background-origin: content-box; /* From content edge */
Origin: Content Box

background-clip

Controls how far the background extends.

Roman Urdu: Background kahan tak dikhana hai? Text ke andar bhi kar sakte hain (cool effect).

background-clip: border-box; /* To border edge (default) */ background-clip: padding-box; /* To padding edge */ background-clip: content-box; /* To content edge */ background-clip: text; /* Clip to text (webkit) */
Text Clipping Effect!

background-blend-mode

Controls how background images blend with each other and background color.

Roman Urdu: Multiple backgrounds ko mix kaise karna hai? Photoshop jaisa blend effects.

background-blend-mode: normal; /* Default */ background-blend-mode: multiply; background-blend-mode: screen; background-blend-mode: overlay; background-blend-mode: darken; background-blend-mode: lighten;
Blend Mode: Multiply

Multiple Backgrounds

CSS supports multiple background layers.

Roman Urdu: Ek saath kai backgrounds laga sakte hain. Layers jaisa - ek ke upar ek.

background: linear-gradient(rgba(255,255,255,0.3), rgba(255,255,255,0.3)), url('pattern.png') center/60px repeat, linear-gradient(45deg, #667eea, #764ba2);
Multiple Background Layers

background (Shorthand)

Combines all background properties in a single declaration.

Roman Urdu: Sab background properties ko ek line mein likhne ka tareeka. Time saving!

/* Syntax: background: [color] [image] [repeat] [attachment] [position] / [size] */ background: #ff6b6b url('image.jpg') no-repeat fixed center/cover; /* Multiple backgrounds */ background: linear-gradient(45deg, rgba(255,0,0,0.5), rgba(0,255,0,0.5)), url('texture.png') repeat, #333;

üí° Tips for Background Shorthand:

  • Order matters! Follow the syntax above
  • The size must come after position, separated by a slash (/)
  • You can omit values you don't need
  • For multiple backgrounds, separate each layer with a comma

üèÜ Best Practices & Performance Tips

üöÄ Performance

  • Optimize image sizes and formats
  • Use CSS gradients instead of images when possible
  • Consider using CSS sprites for multiple small images
  • Use will-change for animated backgrounds

üì± Responsive Design

  • Use background-size: cover for responsive images
  • Test different screen sizes and orientations
  • Consider different images for different breakpoints
  • Use relative units for positioning

‚ôø Accessibility

  • Ensure sufficient contrast between text and background
  • Don't rely solely on background images for important content
  • Provide fallback colors for gradient backgrounds
  • Test with high contrast mode

üé® Design

  • Layer backgrounds thoughtfully (lightest to darkest)
  • Use subtle patterns and textures
  • Consider the content that will overlay the background
  • Test in different lighting conditions
